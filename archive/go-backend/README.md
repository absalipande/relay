# Archived Go Backend Prototype

This folder contains the earlier Relay Go backend prototype.

It is preserved for reference only. The active Relay stack is now:

- Next.js frontend
- Fastify Node.js API
- Supabase Postgres
- Supabase Auth
- Supabase Storage later

Archived contents include:

- Go API entrypoints in `cmd/`
- Go packages in `internal/`
- Goose/Limen migrations in `migrations/`
- Go module files
- old local setup Makefile and env example
- old Go backend docs in `docs/`

Do not extend this prototype unless intentionally reviving the Go stack.
